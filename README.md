# 📰 Dataverse News

Dataverse News is a modern, fully-featured news aggregator website that brings together the latest stories from six different global news sources.

---

## 🚀 SETUP INSTRUCTIONS

```bash
# 1. Clone or unzip the project
cd dataverse-news

# 2. Install dependencies
npm install

# 3. Add your API keys (see .env section below)
cp .env.example .env
# Then edit .env with your actual keys

# 4. Start development server
npm run dev

# 5. Build for production
npm run build
```
